# SteakMinimap
### World of Warcraft: Wrath of the Lich King

![Screenshot showing the Minimap](https://trap-nine.com/screenshots/WoWScrnShot_031126_103100.jpg)

Rework of the default minimap.

* Use hi-res map textures for a larger minimap.
* Automatic quest tracker that shows quests from the current zone.
* Group unit locations on minimap that indicate offline, dead, raid target icon or class-colored dots.
* Learn flight paths and repair vendors.  More points of interest to be added.

This is still very much a work in progress.  I am still adding features and tweaking things to make them work better.
